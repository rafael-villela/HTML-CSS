Modelo de Caixas

Formas de estilizar as caixas:

  • Tamanho: height e width
  • Borda: border
  • Padding: aumentando a caixa em relação a borda
  • Margin: espaço externo para não esbarrar em outra caixa
  • Borda fora da caixa: outline

Tipos de caixa

Modelo de caixas na prática

O modelo de caixas na prática se da pelo conceito de cada tag praticamente se enquadrar no site como uma caixa de conteúdo. Com isso, podemos adaptar esteticamente componentes dessas caixas como margem, borda, pading, outline, como podemos ver na seguinte imagem:

Nesse contexto, para analisar esses membros sem ter que ficar chutando valores, podemos utilizar o inspecionar do google.

Então temos as seguintes shorthands para edição dentro das caixas:

  • Border: width style color
  • Padding: top rigth bottom left
  • Margin: top rigth bottom left
  • Outline: width style color

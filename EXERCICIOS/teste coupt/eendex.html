    Imagem dinâmica

    Tente abrir esse site em vários dispositivos diferentes ou simplesmente aumente e diminua o tamanho do seu navegador

    Para adiconar uma imagem dinâmica utilizamos a tag <picture> e dentro dela colocamos a <img>preferida.

    Com isso, para adaptar a imagem ao tamanho da tela do navegador, usamos as tags <source-media-type>para add imagens que irão aparecer caso o tamanho da tela mude.

    Devemos ter a noção semântica de que as imagens menores devem estar em sources em linhas acmia das imagens maiores

    Colocando áudio no site

    Vamos aprender a reproduzir áudio no site em HTML5

    Para adicionar audio no site, utilizamos a tag <audio> e colocamos o arquivo dentro do src=" ".

    para deixar o audio tocando quando abre o site usa-se o autoplay. No entanto caso não esteja funcionando, deve tentar utilizar outros sources sem ser mp3 como ogg ou wav

    Para add um player para escutar o audio, adicionamos controls após o src=" "

    temos também as funções preload e loop

    Adicionando vídeos no site com hospedagem local

    este vídeo está hospedado no meu próprio servidor

    para adicionar vídeos com hospedagem interna no site, usamos a tag <video> em que podemos add o vídeo e mudar seu tamanho, adicionar um player e também colocar uma capa usando width, controls e poster logo após <video

    é importante colocar através de <source src>outros tipos de arquivo do vídeo para que possam rodar em outros navegadores

    inserindo vídeos do youtube

    para inserir um vídeo do youtube basta ir em compartilhar no vídeo escolhido, clicar em incorporar e copiar o código. Depois basta colar no Virtual Studio Code que já será incorporado ao site
